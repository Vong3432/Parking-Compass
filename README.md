# Parking-Compass
App that helps human save and navigate parked vehicle with Compass and Map.

## Features
**Save and find parking location** 

- Save vehicle parking location.
- Use compass/map to find parking location to vehicle.

**Parking location histories**

- View previous parking location history.

**Access to frequently used parking location**

- Decide and save a frequently parked location to save parking location, and mark it as current parking location directly without opening location manually.
